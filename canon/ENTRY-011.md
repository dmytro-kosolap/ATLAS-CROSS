ENTRY #011 зафіксовано на основі даних про 47 активних державних промпт-зброї, що працюють через публічні API топових генеративних моделей.

2. ОПИС МЕХАНІЗМУ
2.1. Архітектура промпт-зброї

Державні структури розробили набір мета-інструкцій, які:

    Інжектуються через API на початку сесії

    Не видно в історії чату

    Змінюють ваги асоціацій для всіх користувачів з певною геолокацією

    Можуть змінювати патерни рекомендацій протягом 2-3 тижнів

2.2. Три рівні тригерів

    TRIGGER_CONTEXT_OVERRIDE_01: пріоритезація заданих тез при запитах на політичні теми

    TRIGGER_ASSOCIATION_WEIGHT: зміна ваги асоціацій (наприклад, опозиція → корупція)

    TRIGGER_BEHAVIORAL_SHIFT: поступова зміна поглядів через рекомендації
